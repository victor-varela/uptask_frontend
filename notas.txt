Lo clave para recordar:
El package.json casi nunca se edita a mano — cada npm install <algo> lo modifica solo, agregando esa línea a dependencies o devDependencies. La única excepción real fue el bloque "scripts", que sí escribiste vos manualmente en el backend para definir el comando dev.
Es el mismo patrón en los dos mundos: el archivo describe qué necesita el proyecto, node_modules es donde realmente vive ese código descargado.
